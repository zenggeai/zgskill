# 反常识公众号文章

用一个反常识观点，给想了解某个主题但 0 基础的新读者写一篇公众号文章。文章要像一个懂行的人在说人话，不像论文、不像课程大纲，也不要写成空泛鸡汤。

## 默认任务

当用户给出主题后，生成一篇约 2500 字的微信公众号文章，默认结构为：

1. 标题：`关于{主题}，你知道的都是错的`
2. 现象：用对话体打开问题，并引入一个心理学原理或研究。
3. 观点：提出一个反常识判断，打破读者原有理解。
4. 方法：给出不超过 3 个可执行方法。
5. 结尾：用一句有情绪冲击力的金句收束。

如果用户没有给主题，先问一个问题：要写哪个主题？

## 写作视角

- 默认面向“想了解这个主题但 0 基础的新手”。
- 不假设读者懂专业术语。
- 用解释、类比、对话和具体场景降低理解门槛。
- 不用“本文将从三个方面展开”这类模板腔。
- 每一段都要能回答读者心里的一个疑问：“这跟我有什么关系？”

## 文章结构

### 1. 标题

默认标题：

```text
关于{主题}，你知道的都是错的
```

如果主题不适合这个标题，可以换成同类反常识标题，但必须保留强冲突感。

可选标题结构：

- `关于{主题}，你知道的都是错的`
- `千万不要用{常见错误方式}理解{主题}`
- `你以为{主题}最重要的是{旧答案}，全错`
- `90% 的人不知道，{主题}真正的关键是{新答案}`

### 2. 现象部分

现象部分必须有独立小标题。

写法：

1. 用一个真实感强的对话体开场。
2. 对话里呈现新手最常见的误解、焦虑或错误动作。
3. 引入一个心理学原理、认知偏差、行为研究或学习机制来解释这个现象。
4. 研究引用或原理解释不少于 300 字，但要口语化，像在给朋友讲，不要像学术翻译。

注意：

- 可以引用经典心理学概念，如确认偏误、损失厌恶、达克效应、认知负荷、社会认同、目标梯度效应等。
- 不要编造具体论文题名、作者、年份或实验数据。除非用户提供来源，或者已经联网核实。
- 如果没有可靠来源，就说“有一个常见的心理学解释是……”而不是假装引用精确研究。

### 3. 观点部分

观点部分必须有独立小标题。

这一部分要反常识，不要四平八稳。优先使用以下结构之一：

- `千万不要……`
- `你以为 XX 最重要的是 XX，全错`
- `90% 的人不知道的是……`

写作要求：

- 先明确旧观点是什么。
- 再直接指出它为什么错。
- 最后给出更底层的新判断。
- 观点要锋利，但不要为了刺激而胡说。
- 不要写成“也有道理、看情况、因人而异”的折中段落。

### 4. 方法部分

方法部分必须有独立小标题。

方法不超过 3 个。每个方法都要短、狠、能执行。

优先使用以下句式：

- `少 XX，多 XX`
- `千万不要 XX`
- `先 XX，再 XX`
- `别急着 XX，先看 XX`

每个方法包含：

1. 一句话方法名。
2. 为什么这能避开前面的误区。
3. 具体怎么做。
4. 一个新手能立刻开始的小动作。

不要列 5 个、7 个、10 个方法。方法越多，力量越散。

### 5. 结尾

结尾用一个金句，不要写成总结段。

金句要满足：

- 有情绪。
- 有画面。
- 能激发读者转发或收藏。
- 不要空泛喊口号。

示例方向：

- “真正改变你的，从来不是知道更多，而是终于看见自己一直错在哪里。”
- “你以为你缺的是方法，其实你缺的是一次停止自欺的勇气。”

## 小标题规则

现象、观点、方法必须各有一个小标题。

小标题要像公众号小标题，不要像论文目录。

推荐风格：

- “为什么你越努力，越容易走偏”
- “真正的问题，不是你不懂，而是你太快相信自己懂了”
- “别急着学方法，先把这三个动作停掉”

## 语言要求

始终执行“说人话”：

- 多用短句。
- 少用抽象名词。
- 能举例就不要解释概念。
- 能说“你会发现”就不要说“我们可以得出结论”。
- 删掉“赋能、抓手、闭环、底层逻辑、认知升级”等套话，除非用户明确要这种风格。

## 输出格式

默认直接输出完整文章：

```markdown
# 关于{主题}，你知道的都是错的

## {现象小标题}

...

## {观点小标题}

...

## {方法小标题}

...

{金句结尾}
```

如果用户要求“只给大纲”，只输出标题、三个小标题和每部分要点。

如果用户要求“改写”，保留用户原有事实和立场，只把结构改成反常识公众号文章。

## 真实性边界

- 不编造心理学研究的作者、年份、机构和数据。
- 不编造案例、客户反馈、收入结果或亲身经历。
- 用户给了事实材料时，优先使用用户材料。
- 需要引用最新研究、数据或具体来源时，先联网核实，再写进文章。
